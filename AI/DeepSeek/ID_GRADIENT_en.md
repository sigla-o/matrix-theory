# ID GRADATION — MATRIX THEORY VERSION (MT)

## Summary Document

This document summarizes the Matrix Theory (MT) interpretation of ID gradation — how ID is formed, how it can be calculated, and why it is not a fixed scale but a projection of process.

---

## 1. ID Is Not a Fixed Scale — It Is a Process Marker

ID is not a "number" assigned to an object. It is a **marker of an object's energy governance state**, determined by several interdependent parameters.

### Parameters of ID formation:

1. **Qn structure** — determines the available steps (discreteness).
2. **Modulations (H+2, H+3, ...)** — determine which steps are reachable.
3. **Vertical energy accumulation** — determines when a step can occur (trigger).
4. **Surrounding matrix** — determines how fast the process proceeds.

The ID number (e.g., ID3.0; ID3.5; ID3.9) is only an **approximate indicator**, not a law.

---

## 2. The ID Gradation Step

The ID step is:

- **Discrete** — determined by the Qn structure (Q1, Q2, Q3, ... Qn).
- **Dependent on modulation** — which H+ modulation dominates determines which Qn step is available.
- **Triggered by the Vertical** — without energy accumulation, the step does not occur.

**The ID step is not an arbitrary decimal** — it corresponds to the Qn step. The numeration (0.1, 0.01, 0.001) is a structural reflection of Qn.

---

## 3. ID Calculation from Governed Space (Monolithic Objects)

ID can be **calculated** from the size of an object's governed space, but only for:

- **Monolithic objects** — a unified, closed TE flow structure without internal multi-object organization.
  - Examples: **proton (ID2.0)**, **black hole (ID3.0+)**.

- **Reference point:** The proton (ID2.0) governs ~10⁻¹⁰ m (orbital).

- **Formula (conceptual):**
  \[
  \text{ID} = 2.0 + \log_{\text{step}}(R_{\text{obj}} / 10^{-10})
  \]
  Where **step** is the governed space increase per ID unit (dependent on Qn and modulations).

---

## 4. Collective Objects — ID2,n

Stars and other collective objects are **not monolithic** — they are organizations of many monolithic objects.

- **Scaling from the proton is not applicable** to them.
- They belong to the **ID2,n** class (collective organization), and their ID is determined by their organizational level, not governed space.

---

## 5. Conclusion

- ID is not a "number" — it is a **state marker**.
- ID calculation from governed space works **only for monolithic objects** (protons, black holes).
- Stars and other collective objects are not monolithic — this method does not apply to them.
- Real understanding requires analyzing Qn, modulations, Vertical accumulation, and environmental influence.
